# agile-task-manager
Projeto acadêmico de Engenharia de Software com metodologias ágeis


# Construindo um projeto Ágil no git hub

Este projeto Simula o desenvolvimento de Sistema de gerenciamento de tarefas utilizando metodologias Ágeis, com foco em organização do trabalho, controle de qualidade e gestão de mudanças.

O projeto faz parte da disciplina de Engenharia de Software e utiliza o Github como ferramenta principal para versionamento, planejamento e intergração contínua.

## Planejamento
O projeto utiliza em quadro Kanban no github Projects para acompanhamento das tarefas
